Verlan is a type of argot in the French language, featuring inversion of syllables in a word, and is common in slang and youth language.

Examples:

+----------+---------+---------+
| original | verlan  | meaning |
+----------+---------+---------+
| bande    | deban   | group   |
| black    | kebla   | African |
| bonjour  | jourbon | Goodday |
| classe   | secla   | class   |
| femme    | meuf    | woman   |
| flic     | keuf    | cop     |
+----------+---------+---------+
